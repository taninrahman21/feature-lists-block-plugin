The Feature Lists WordPress Block Plugin provides an intuitive and flexible way to showcase features in a visually appealing list format.

== Description ==
The Feature Lists WordPress Block Plugin is an innovative and user-friendly tool designed to enhance the way you showcase features on your WordPress site. Seamlessly integrated with the Gutenberg editor, this plugin empowers users to create visually engaging feature lists with minimal effort, making it an ideal solution for highlighting product features, service offerings, or key benefits in an attractive list format.


**[Demos](https://bblockswp.com/demo/block-name)**


= Features =
-**Custom Icons**: Select from a range of icons or upload your own to represent each feature visually.
-**Rich Text Editing**: Use the Gutenberg editor's powerful text formatting tools to style your feature titles and descriptions.
-**Responsive Design**: Ensures your feature lists look great on all devices, from desktops to mobile phones.
-**Styling Options**: Customize the look and feel of your feature lists with various styling options, including color choices and layout settings.
-**Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First, install the B Feature Lists Block plugin
- Add the B Feature Lists Block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **B Feature Lists**
3. Click on the **B Feature Lists** to add the block

= Download & Upload: =
1. Download the **B Feature Lists** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`plugin-slug.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **B Feature Lists Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
